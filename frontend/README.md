# To-Do List

Aplicação web simples de lista de tarefas desenvolvida com React e Vite.

## Funcionalidades
- Adicionar tarefas
- Marcar tarefas como concluídas
- Deletar tarefas

## Como rodar
1. Instale as dependências:
```bash
npm install
```
## Rode o servidor de desenvolvimento:
```
npm run dev
```
## Abra no navegador 
```
http://localhost:5173
```

## Tecnologias
```
React
Vite
JavaScript
HTML/CSS
```
